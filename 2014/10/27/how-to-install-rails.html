如何安装Rails

前期准备

我们讲主流的操作系统分成三种：Windows、Mac OS、Linux。

Rails作为一个使用Ruby语言编写的开发框架，依赖于Ruby的运行环境。

Mac OS 系列

首先安装Xcode，从苹果官方的App Store下载即可。

Xcode中包含编程所需的各类基本组件，如gcc、libxml2等。正确安装并配置Xcode是在Mac环境下编程的基本要素。

安装完成后，进入命令行工具Terminal，安装homebrew工具，执行以下命令：

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

HomeBrew是Mac操作系统下的软件包管理系统，在brew的软件源中可以找到绝大多数的运行包、

Windows 系列

下载`Virtual

Box`软件并安装。

安装Vagrant，点击此处.

安装MinGW, 勾选所有包并安装。

打开命令提示行或PowerShell，通过命令打开一个你喜欢的文件夹，在其中运行以下命令：

mkdir dev
cd dev
vagrant init precise64 http://files.vagrantup.com/precise64.box
vagrant up

首次运行会把一个已经打包好的Ubuntu 12.04.2 LTS下载到本地，可能会花一些时间。

运行完成后运行：

vagrant ssh

此时若出现类似：

$ vagrant@precise64:~#

则说明成功进入虚拟机。

Vagrant是一个用Ruby开发的虚拟机控制系统，可以方便地联系Windows和虚拟机内的文件系统。保存有vagrant的文件夹中的所有内容都会在虚拟机中对应/vagrant目录中找到。在Windows文件系统中，Vagrantfile是Vagrant的配置文件，可以在其中编辑该vagrant虚拟机的配置。

Linux 系列

无需进行前期配置。

安装Ruby

在后文中，Windows平台中的命令若不特别说明，一律在vagrant ssh中使用 Linux 的命令。

在后文中，Linux平台以Ubuntu/Debian系列作为范例，Fedore/CentOS/RHEL需要将apt-get换为yum，部分软件包名称有所区别。

Linux平台需要首先需要安装Ruby所需的依赖包：

sudo apt-get update && apt-get upgrade -y
sudo apt-get install gcc g++ curl wget ssh git-core autoconf -y

安装Ruby版本控制器RVM

curl -L https://get.rvm.io | bash -s stable

调整RVM源为国内源：

#重启或重启terminal后

# FOR MAC
sed -i .bak 's!cache.ruby-lang.org/pub/ruby!ruby.taobao.org/mirrors/ruby!' $rvm_path/config/db

# FOR LINUX
sed -i 's!cache.ruby-lang.org/pub/ruby!ruby.taobao.org/mirrors/ruby!' $rvm_path/config/db

安装Ruby

rvm install 2.1.3

调整RubyGem的源到国内：

gem sources --remove https://rubygems.org/
gem sources -a https://ruby.taobao.org/

安装Rails

gem install rails

搞定~
